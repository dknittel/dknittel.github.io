Using version control is beneficial because it allows the user to share their work with other people and allows multiple people to work on the same file at the same time.

Git helps you keep track of changes by making a note of every time you change the state of a file. It also allows you to return to the most recent commit.

It is helpful to use GitHub to store code because it allows you to save your work on the internet. Therefore, you can access your file from any computer. Additionally, anyone else on your team can access that file and make changes even while you are making changes.
